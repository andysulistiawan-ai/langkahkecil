# langkahkecil 🌱

**Langkah kecil menuju kebiasaan baik** — Small steps towards good habits.

Multi-tracker harian: To-Do List, Financial Tracker, Weight Tracker.

## Fitur

- ✅ **To-Do List** — Tambah, edit, hapus, reorder, color tags, focus timer
- ✅ **Keuangan** — Catat pemasukan/pengeluaran, grafik tren 7 hari, filter
- ✅ **Berat Badan** — Log berat, grafik tren, kalkulator BMI
- ✅ **Pengaturan** — Tema gelap/terang, bilingual (ID/EN), sync Google Sheets
- ✅ **Offline-first** — Semua fitur bekerja tanpa internet

## Tech Stack

React 18+ • TypeScript • Tailwind CSS • Zustand • Recharts • Vite

## Development

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # Production build → dist/
npm run preview  # Preview production build
```

## Google Sheets Sync

1. Buat Google Sheet baru
2. Extensions → Apps Script → paste Code.gs (download dari Settings)
3. Deploy → Web App → copy URL
4. Paste URL di Settings → Sync
